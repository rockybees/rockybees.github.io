Exponential Notation

An $\textbf{exponent}$ indicates how many times the base is used as a factor.

$3^4 = 3 \times 3 \times 3 \times 3$ can be read as "three to the power of four", or "three to the fourth power".

Exponent Rules Summary Mathematical Form
Product Rule \( a^m \times a^n = a^{m+n} \)
Quotient Rule \( \frac{a^m}{a^n} = a^{m-n} \)
Power Rule \( (a^m)^n = a^{m \times n} \)
Zero Exponent Rule $ a^0 = \frac{a^m}{a^m} = 1$, for any $a \neq 0$
Negative Exponent Rule \( a^{-n} = \frac{1}{a^n} \)
Fractional Exponent Rule \( a^{\frac{m}{n}} = \sqrt[n]{a^m} \)
$\mathbb W = \{ 0, 1, 2, \ldots \}$

Note: For exponents of $0$, $0^0$ is undefined or indeterminate because different mathematical contexts suggest different values.

If we apply the above exponent rule to $0$, it would suggest $0^0 = 1$.

However, if we extend the exponent rule $0^n = 0$ (for any $n>0$) to $n = 0$, it would imply $0^0 = 0$, which contradicts the first rule.
