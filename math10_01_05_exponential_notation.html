Exponential Notation

An $\textbf{exponent}$ indicates how many times the base is used as a factor.

$3^4 = 3 \times 3 \times 3 \times 3$ can be read "three to the power of four", or "three to the fourth power".

Exponents of $0$:

$a^0 = 1$, for any $a \neq 0$.

Note: $0^0$ is undefined or indeterminate because different mathematical contexts suggest different values.

If we apply the above exponent rule to $0$, it would suggest $0^0 = 1$.

However, if we extend the exponent rule $0^n = 0$ (for any $n>0$) to $n = 0$, it would imply $0^0 = 0$, which contradicts the first rule.


Example 1:

Find the positive square root of 144.

Solution:

$144 = 2 \times 2 \times 2 \times 2 \times 3 \times 3$,

Therefore, $\sqrt{144} = 2 \times 2 \times 3 = 12$.

Mixed Root and Entire Root:

An expression such as $3\sqrt{2}$ is called $\textbf{mixed root}$, and the expression $\sqrt{18}$, which equals $3\sqrt{2}$, is called an entire root.

Any mixed root can be converted into an entire root. E.g., $3\sqrt[3]{2} = \sqrt[3]{54}$
